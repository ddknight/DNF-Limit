WinXP、Win7版本：基于.net 2.0编译
Win8、Win10版本：基于.net 4.5编译
两者功能并没有本质区别，如果同时安装.net2.0及.net4.5，无需区分版本均可使用

关于.net运行库
1、2.0为独立安装包，3.5包含2.0，因此装3.5运行库之后无需独立安装2.0运行库
2、4.5及以上版本，均向下兼容（装有4.7运行库，可以运行基于4.5编译软件，不可直接运行基于3.5编译的软件）
3、Win7自带3.5运行库（可运行2.0）；win10自带4.5运行库（运行2.0、3.5需安装3.5运行库）
4、.net运行库下载地址:https://dotnet.microsoft.com/download/dotnet-framework
5、XP用户建议使用.net2.0版本，下载地址https://www.microsoft.com/en-us/download/details.aspx?id=1639
